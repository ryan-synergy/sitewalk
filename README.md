# SiteWalk

AV site-survey field tool. Single-file web app — walk a home, capture rooms (TV, audio, sources, control, wiring), network, cameras, and head-end; export CSV / JSON / printable summary for quoting.

- **Use it:** open the GitHub Pages URL on iPhone/iPad → Safari Share → **Add to Home Screen**. Works offline after first load.
- All survey data stays on the device (localStorage). Back up with the JSON export; restore with Import.
- Stack: one HTML file, no build step. `sw.js` provides offline caching.

Source of truth for design decisions and product spec lives in the AgentWorkSpace conversation history / memory.
